# Barnet FC Business Analysis Workshop

An interactive web application designed for conducting hands-on Business Analysis workshops using a realistic scenario involving Barnet FC's planned return to their home borough through stadium development.

## üéØ Workshop Overview

This workshop provides practical experience with key Business Analysis frameworks through an engaging football club scenario. Participants work through the complexities of Barnet FC's ¬£25 million, 3-year stadium relocation project, applying structured analysis techniques to real-world challenges.

### Learning Objectives
- Apply PESTLE analysis to identify external macro-environmental factors
- Conduct comprehensive stakeholder analysis and mapping
- Perform CATWOE analysis from multiple stakeholder perspectives
- Experience collaborative problem-solving in a realistic business context
- Understand how different BA frameworks interconnect and inform decision-making

### Workshop Sections

- Local residents and community groups
- Club management and supporters
- Local government officials
- Business owners and environmental officers
- League representatives and construction professionals

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely client-side

### Installation & Usage
1. Download or clone the project files
2. Open `index.html` in your preferred web browser
3. Navigate through the workshop sections using the sidebar
4. Follow the structured activities and timing guidelines
5. Use the timer feature to manage workshop pacing

### Workshop Facilitation
- **Duration**: Approximately 85 minutes total
- **Team Size**: 3-5 participants per team recommended
- **Materials Needed**: Flip charts or digital whiteboard for team work
- **Preparation**: Review facilitator notes in each section
